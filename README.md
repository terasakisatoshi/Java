# Iterator パターンにつて
* Iterator パターンとは，何かたくさん集まっている時に，
それを順番に指し示して，全体をスキャンしていく処理を行うもの．
* iterator は日本語では反復子と呼ばれることがある．

＃Adapter　パターンについて
* 継承と委譲を用いたパターンが存在するようだ．
* 正直使いどころがピンとこないので先に進みます．
